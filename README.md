Dendê-statistics 🌴📊
Este projeto é um template base para a Primeira Atividade da disciplina de Mineração de Dados, ministrada no Centro Universitário de Excelência (UNEX) — Unidade Feira de Santana.

O objetivo do projeto é apoiar os estudantes no desenvolvimento de conceitos fundamentais de Estatística Descritiva, aplicados à análise de dados, utilizando a linguagem Python e boas práticas de programação orientada a objetos.

📚 Contexto Acadêmico
Este projeto foi idealizado pelo professor Lucas Almeida Silva como parte das atividades práticas da disciplina de Mineração de Dados.

A proposta é que os estudantes implementem, testem e validem métodos estatísticos clássicos, compreendendo não apenas o uso das fórmulas, mas também sua aplicação prática em conjuntos de dados reais ou simulados.

🏢 Sobre a Dendê SoftHouse (Empresa Fictícia)
A Dendê SoftHouse é uma empresa fictícia de desenvolvimento de software, criada exclusivamente para fins acadêmicos nas atividades da universidade.

Ela é responsável pelo desenvolvimento do aplicativo Dendê Eventos, uma iniciativa local voltada para:

Divulgação de eventos culturais, acadêmicos e sociais
Gestão de eventos (datas, categorias, público-alvo, frequência)
Apoio à tomada de decisão por meio da análise de dados
Os dados analisados neste projeto simulam informações que poderiam ser coletadas pelo aplicativo Dendê Eventos, servindo como base para os estudos de mineração e análise estatística.

🎯 Objetivo do Projeto
Implementar uma classe Statistics capaz de realizar cálculos estatísticos fundamentais, incluindo:

Média
Mediana
Moda
Variância
Desvio padrão
Covariância
Frequências absoluta, relativa e acumulada
Probabilidade condicional
Quartis
Histogramas por buckets
Todos os métodos devem ser implementados sem o uso de bibliotecas estatísticas prontas, reforçando o aprendizado conceitual.
